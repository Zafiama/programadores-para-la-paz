# Semana6 - Práctica

Instrucciones para ejecutar:
1. En la raíz del repositorio, instalar dependencias:
   npm install

2. Ir a la carpeta Semana6 y ejecutar el servidor:
   npm start

Rutas disponibles:
- POST http://localhost:3000/registro
- POST http://localhost:3000/incidencia

Archivos incluidos en /Semana6:
- server.js
- prueba-api.txt
- ejemplo-incidencia.txt
- reflexion-semana6.txt
- Preguntas-semana6.txt
- package.json
